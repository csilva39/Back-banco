Antes de ejecutar el proyecto, asegúrese de tener instaladas las siguientes herramientas:

Node.js versión 18 o superior
PostgreSQL versión 14 o superior
Un cliente como pgAdmin o DBeaver (opcional, para revisar la base de datos)

Dentor del rpoeycto instale las dependencias: 

npm install

***************
En el index de la carpeta db confiure credencuales d ela base de datos para la conexión 

Cree la base de datos en PostgreSQL (por ejemplo, con nombre minibanco_db).

Y ejecute 

npm start
